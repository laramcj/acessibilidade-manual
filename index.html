Hand Talk

O Hand Talk é um aplicativo gratuito para celulares Android e iPhone (iOS) que permite traduzir textos em Libras (Língua Brasileira de Sinais). A ferramenta usa um avatar digital, interpretado pelo personagem Hugo, para desenvolver os gestos e facilitar a comunicação com pessoas surdas ou com dificuldade auditiva.

Rybená

Rybená é um aplicativo para dispositivos móveis com Android ou iOS, capaz de ajudar na comunicação com pessoas que possuem deficiência auditiva, além de ler mensagens para deficientes visuais. Desenvolvido pelo Grupo ICTS, de comunicação acessível e inclusão digital, o app permite realizar, de forma rápida e prática, a tradução de palavras e frases em português para libras, a linguagem brasileira de sinais.

Leitores de Tela

Leitores de tela são softwares usados para obter resposta do computador por meio sonoro, usado principalmente por deficientes visuais. O programa vai percorrendo textos e imagens e lendo em voz alta tudo o que ele encontra na tela, assim como as operações que o usuário realiza com as teclas alfanuméricas e os comandos digitados.

A navegação por leitores de tela funciona essencialmente de três maneiras:
1. Lendo toda a página (navegação com as setas);
2. Lendo os links (navegação com a tecla Tab);
3. Lendo os cabeçalhos (navegação com a tecla h);
A partir desses três comandos principais, o usuário consegue se guiar pela estrutura da página enquanto vai acompanhando a fala humana emitida pelo software ao ler seu conteúdo.
Por isso mesmo é importante que tanto o design quanto o código estejam prontos para receber o usuário que navega utilizando leitores de tela. Um simples cabeçalho quebrado ou mal implementado pode fazer com que o usuário se perca na página e tenha dificuldades em entender como a informação está organizada hierarquicamente.
Estes são alguns dos softwares mais comuns:
Jaws: leitor de tela pago para Windows
Virtual Vision: leitor de tela pago para Windows
NVDA: leitor de tela gratuito para Windows
Window-Eyes: software pago para Windows
Orca: leitor de tela gratuito para Linux
VoiceOver: Leitor de tela para IOS que acompanha os dispositivos da Apple

Be My Eyes

O Be My Eyes é uma plataforma criada para que pessoas com deficiência visual e voluntários possam se conectar através de videoconferências. Escolhido pelo Google como um dos melhores apps de 2018, essa ferramenta permite que pessoas "emprestem" seus olhos para deficientes visuais em conversas de áudio e vídeo no celular. O app está disponível para Android e iPhone (iOS).

Texto-Audio

Balabolka



aplicativo Balabolka permite que você transforme qualquer texto em áudio em poucos cliques.Você pode alterar a velocidade da voz e o tom, assim como a própria voz. O Balabolka inclui dois oradores padrão (homem e mulher), apesar de poder utilizar qualquer outra voz armazenada no seu computador.




Transcriber for WhatsApp



O Transcriber for WhatsApp é um aplicativo gratuito para celulares Android que converte as mensagens de áudio recebidas via WhatsApp em texto. O recurso é importante para quem tem problemas de audição.




Textr - Voice Message to Text



O app Textr – Voice Message to Text pode converter facilmente áudios do WhatsApp para texto. É um aplicativo muito útil para pessoas com deficiência auditiva, ou para quem está em lugares onde não dá para ouvir os áudios.


Texto alternativo em imagens

Todas as informações de uma página acessível devem ser apresentadas em texto. Isso significa que, se for usada alguma outra mídia, como imagens e sons, as informações que elas contêm devem ser repetidas numa descrição textual. Essa descrição deve ser "equivalente", isto é, deve transmitir as mesmas informações que os elementos disponibilizados.

-IMAGENS COM CONTEÚDO

Fotos, gráficos, organogramas, ilustrações, imagens que substituem botões ou links, gráficos etc.. Todas as imagens que apresentam conteúdo devem conter uma descrição. A descrição da imagem pode ser informada no próprio texto ou por meio de um atributo específico do elemento img no HTML, também chamado de ALT. A descrição fornecida através deste atributo não aparece visualmente. Esse recurso só será reconhecido pelo leitor de ela quando o usuário, navegando pelo teclado, passar por ela. Desta forma, os usuários com deficiência visual sabem qual é o conteúdo da imagem por meio do texto de descrição.


- IMAGENS DECORATIVAS

Utilizadas para decorar a página ou partes dela. Como exemplos, podemos citar marcadores de lista estilizados, cantos arredondados, ícones utilizados para decorar ou enfatizar um link, etc.. As imagens decorativas devem ser ignoradas pelos recursos de tecnologia assistiva. Ou seja, os leitores de tela não devem considerar a presença delas, pois não há conteúdo a ser transmitido. Para que o código esteja preparado, a recomendação é a inserção de todas as imagens decorativas através das folhas de estilo (CSS). Em relação ao atributo ALT, a indicação é não preenchê-lo com nenhuma informação, também chamada de alt vazio (alt=""). Assim, os leitores de tela não reconhecerão esses conteúdos, facilitando a navegação assistiva.

Navegação de teclado

A navegação exclusivamente pelo teclado é usada por pessoas que possuem deficiência visual ou motora (para as que possuem baixa ou nenhuma destreza/coordenação motor).

Existem 3 tipos de atalhos para navegação que podem ser usadas individualmente ou mais de uma ao mesmo tempo.

  • Teclas de atalho do próprio navegador;
  • Teclas de utilização do leitor de tela ou outras tecnologias assistivas;
  • Teclas de navegação e atalho de um site criadas por seu desenvolvedor.
